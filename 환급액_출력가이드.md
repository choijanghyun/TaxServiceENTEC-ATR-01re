# 국세청 홈택스 경정청구 환급액 출력 가이드

본 문서는 경정청구 환급액 극대화 AI 점검 프롬프트의 분석 결과를 납세자 관점에서 알기 쉽게 정리하여 제출하기 위한 최종 레포트 출력양식의 작성 가이드입니다.

> 2026년 2월 기준 | 조세특례제한법 및 국세기본법 반영 | 25개 점검 항목 대응

# 문서 구성 안내

본 가이드는 AI 점검 결과를 7개 섹션의 최종 레포트로 구성하는 방법을 안내합니다. 각 섹션별 작성 목적, 포함 항목, 작성 요령, 그리고 제출용 JSON 데이터 구조를 포함합니다.

| 섹션 | 레포트 구성 항목 | 핵심 내용 |
|:---:|---|---|
| A | 해당 가능성 진단 요약 | 납세자 기본 정보 + 경정청구 자격 + 중소기업 판단 결과 |
| B | 후보 항목 리스트 | 적용 가능 공제/감면별 근거, 조건, 필요서류 상세 |
| C | 최적 조합 및 상호배제 검증 | 그룹 A vs B 비교 + 중복 적용 규칙 교차검증 |
| D | 예상 환급/추징 금액 범위 | 최저한세, 농특세, 환급가산금 반영 실질 환급액 |

| F | 주의사항 및 사후관리 | 고용 유지, 자산 처분 추징, 추가 확인 필요 항목 |

# 섹션 A. 해당 가능성 진단 요약

납세자의 기본 정보와 경정청구 자격을 한눈에 파악할 수 있도록 정리하는 섹션입니다.

## A-1. 납세자 기본 정보 카드

AI 점검 프롬프트에 입력된 납세자 기본 정보(No.1~10)를 요약 카드 형태로 표시합니다.

| 사업자 유형 | [개인사업자 / 법인] | 기업 규모 | [소기업 / 중기업 등] |
|---|---|---|---|
| 업종 (KSIC) | [코드 + 업종명] | 사업장 소재지 | [시/군/구 + 수도권 구분] |
| 대상 과세연도 | [YYYY년] | 경정청구 사유 | [미적용 공제 신규 등] |
| 산출세액 | [원 단위] | 과세표준 | [원 단위] |

## A-2. 경정청구 자격 진단

경정청구 기한(국세기본법 제45조의2) 충족 여부와 중소기업 해당 여부(점검 9-2)를 판정합니다.

| 진단 항목 | 판정 결과 | 근거 / 비고 |
|---|---|---|
| 경정청구 기한 (5년 이내) | [적격 / 기한도과] | 국세기본법 제45조의2 제1항 |
| 중소기업 해당 여부 | [해당 / 미해당 / 졸업유예] | 조특법 시행령 제2조 |
| 소기업 / 중기업 구분 | [소기업 / 중기업] | 중소기업기본법 시행령 별표3 |
| 수도권 구분 | [과밀억제 / 성장관리 / 자연보전 / 비수도권] | 수도권정비계획법 제2조 |

> **작성 요령**
> - 판정 결과는 '적격/미해당/추가확인필요' 3단계로 표시합니다.
> - 중소기업 졸업유예(3년)에 해당하는 경우 유예 종료 연도를 명시합니다.
> - 수도권 과밀억제권역 판단은 이후 모든 감면/공제 적용에 영향을 미치므로 정확히 기재합니다.
