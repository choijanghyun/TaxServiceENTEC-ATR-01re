# TaxServiceENTEC-ATR-01re

1. 법인세환급액계산 파일을 마크다운 형태의 번호매김을 적용하여 새로 만들어줘
2. 종소세환급액계산 파일을 4개 구간으로 나누어 병렬로 마크다운 형태의 번호매김을 적용하여 새로 만들어줘

3. 법인세환급액계산, 종소세환급액계산 각각 몇개 Section으로 구성되어 있나?


4. 법인세환급액계산 Section 1~6에 대하여 순차적으로 법제처 국가법령정보(www.law.go.kr) 또는 국세청 법령정보시스템(taxlaw.nts.go.kr) 등 세금 환급금 계산 법률 지식을 딥서치 하여 각 Section별로 오류사항이나 누락된 부분이 발견되면 정확도를 높이기 위하여 점검방식을 달리하여 재검증한 결과도 동일하면 비교 후 정당한 지적사항에 대하여 반영여부 묻지말고 자동으로 반영해줘. 그리고 법인세환급액계산 모두 Section을 점검이 완료되면 종합소득세환급액계산 Section 1~6도 순차적으로 동일하게 수행해줘
