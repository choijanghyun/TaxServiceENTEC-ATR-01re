1. 법인세환급액계산 파일을 4개 구간으로 나누어 병렬로 마크다운 형태의 번호매김을 적용하여 새로 만들어줘

2. 종소세환급액계산 파일을 4개 구간으로 나누어 병렬로 마크다운 형태의 번호매김을 적용하여 새로 만들어줘

3. 법인세환급액계산, 종소세환급액계산 각각 몇개 Section으로 구성되어 있나?

4. deep-research: 법인세환급액계산 Section 1~6에 대하여 순차적으로 법제처 국가법령정보(www.law.go.kr) 또는 국세청 법령정보시스템(taxlaw.nts.go.kr) 등 세금 환급금 계산 법률 지식과 교차검증하여 각 Section별로 오류사항이나 누락된 부분이 발견되면 검증 후 정당한 지적사항에 대하여 반영여부 묻지말고 자동으로 반영해줘. 그리고 법인세환급액계산 점검이 완료되면 종합소득세환급액계산 Section 1~6도 순차적으로 동일하게 수행해줘

5. 법인세환급액계산와 종합소득세환급액계산 2개를 1개로 통합하돼 내용이 많으니깐 Section 1~6에 대하여 순차적으로 Section별로 누락없이 내용을 통합해줘, 주의사항은 각각의 항목에 적용되는 법조항이 중요함으로 누락없이 기재해줘

6. 서브에이전트를 활용하여 gap-detector: Section 1~6 각각의 법인세환급액계산의 내용 대비 통합환급액계산 커버리지 확인과 gap-detector: Section 1~6 각각의 종합소득세환급액계산의 내용 대비 통합환급액계산 커버리지 확인을 병렬 수행해줘

7. deep-research: 법제처 국가법령정보(www.law.go.kr) 또는 국세청 법령정보시스템(taxlaw.nts.go.kr) 등 세금 환급금 계산 법률 지식을 습득하고 통합환급액계산 파일에 누락된 내용이 있는지 교차검증해줘
